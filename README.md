# 🔢 Counter App

A simple and interactive **Counter Web Application** built using **HTML, CSS, and JavaScript**.
Users can easily **increase, decrease, and reset the count value** with a clean responsive UI.

🌐 **Live Demo:** https://counter-45co.vercel.app/

---

## 📌 About

This project is a beginner-friendly JavaScript app that demonstrates **DOM manipulation, event handling, and dynamic UI updates**.

It is useful for learning:

* JavaScript basics
* Button click events
* DOM updates
* UI state management

---

## ✨ Features

* ➕ Increment counter
* ➖ Decrement counter
* 🔄 Reset counter
* ⚡ Instant UI updates
* 📱 Responsive layout
* 🎨 Clean modern design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
